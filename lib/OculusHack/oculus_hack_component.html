Hack Rice 2016: No Man's Forest

For HackRice 6, a friend and I made a procedurally-generated Oculus Rift game. It was a blast to actually get working, so much so that I decided to port it to Mac! The installation isn't general yet (some of the library locations are hardcoded to my machine), but feel free to open a Github issue or pull request about that. I'm teaching myself WebGL so I can port to this website too!
